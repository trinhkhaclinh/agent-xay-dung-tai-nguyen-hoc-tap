## Slide 6 — [bullets] Kiến trúc Client - API Server - Database
*Phần: 2. Sơ đồ kiến trúc*
- Client (Frontend): gửi yêu cầu HTTP và nhận dữ liệu JSON.
- API Server (FastAPI): trung tâm tiếp nhận, kiểm tra dữ liệu và xử lý logic.
- Database (MySQL): lưu trữ và bảo toàn thông tin hệ thống.
**Sơ đồ:** Sơ đồ Client ↔ API Server ↔ Database liên kết bằng mũi tên hai chiều

## Slide 7 — [code] Cấu trúc dự án FastAPI tiêu chuẩn
*Phần: 2. Sơ đồ kiến trúc*
- my_project/
├── app/
│   ├── main.py (Khởi chạy)
│   ├── models.py (ORM Models)
│   ├── schemas.py (Pydantic Schemas)
│   └── routers/ (Định tuyến API)
└── requirements.txt (Thư viện)
