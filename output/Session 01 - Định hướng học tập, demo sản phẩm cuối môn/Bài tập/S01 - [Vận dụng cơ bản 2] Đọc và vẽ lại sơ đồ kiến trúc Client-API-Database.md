# Phân tích sơ đồ luồng dữ liệu 3 lớp

## 1. Bối cảnh nghiệp vụ

Khi phát triển phần mềm, việc hiểu rõ đường đi của dữ liệu từ giao diện người dùng đến database giúp chúng ta dễ dàng thiết kế cấu trúc API phù hợp và xử lý sự cố nhanh chóng.

## 2. Yêu cầu bài toán

Hãy tự vẽ lại sơ đồ kiến trúc Client-API-Database minh họa rõ các bước từ lúc người dùng nhấn nút đăng ký học, cho đến khi database MySQL lưu trữ thông tin sinh viên thành công.

## 3. Yêu cầu đầu ra

- Vẽ sơ đồ luồng bằng công cụ online (draw.io / Lucidchart) hoặc vẽ tay.
- Giải thích rõ vai trò của FastAPI và Database MySQL trong sơ đồ.
- Mô tả định dạng dữ liệu (JSON) truyền tải giữa Client và API.
