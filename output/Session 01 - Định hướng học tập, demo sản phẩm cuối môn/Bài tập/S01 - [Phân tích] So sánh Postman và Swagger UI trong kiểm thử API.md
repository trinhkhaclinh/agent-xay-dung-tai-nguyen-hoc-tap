# So sánh các công cụ kiểm thử API phổ biến

## 1. Bối cảnh nghiệp vụ

FastAPI tự động sinh tài liệu Swagger UI tại đường dẫn `/docs`. Tuy nhiên, trong phát triển dự án thực tế, các lập trình viên vẫn sử dụng thêm công cụ Postman để kiểm thử API nâng cao.

## 2. Yêu cầu đầu ra

### (1) Lập bảng so sánh tính năng

| Tiêu chí so sánh | Swagger UI tự động | Postman Desktop |
| --- | --- | --- |
| Khởi tạo dữ liệu kiểm thử |  |  |
| Hỗ trợ cấu hình biến môi trường |  |  |
| Khả năng lưu trữ lịch sử request |  |  |
| Tự động cập nhật khi sửa code |  |  |

### (2) Đánh giá tình huống

- Đưa ra nhận định: khi nào nên ưu tiên dùng Swagger UI, khi nào nên dùng Postman.
- Giải thích vai trò của việc tạo Collection trong Postman.
