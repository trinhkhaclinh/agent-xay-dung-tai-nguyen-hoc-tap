FastAPI Validation Roadmap & Cheat Sheet

Quy Trình Xử Lý Dữ Liệu Vào (Validation Pipeline)
1
FastAPI Router Phân Tích Đường Dẫn

Khi có Request đến, FastAPI khớp URL mẫu (ví dụ: /students/{student_id}). Nó tự động trích xuất các tham số từ Path hoặc Query và chuyển vào luồng xác thực.

2
Pydantic Ép Kiểu Dữ Liệu (Type Coercion)

Dựa trên Type Hints của Python (int, float, bool), Pydantic chuyển đổi dữ liệu dạng chuỗi từ HTTP thành kiểu Python mong muốn (ví dụ: chuỗi "12" thành số nguyên 12).

3
Kiểm Tra Ràng Buộc (Constraints Evaluation)

Kiểm tra các quy tắc nâng cao được định nghĩa thông qua Query(), Path(), hoặc Field() như độ dài chuỗi tối thiểu min_length, hoặc khoảng giới hạn số học gt, ge, lt, le.

4
Tự Động Trả Về Lỗi 422 Hoặc Chuyển Tiếp Sang Logic

Nếu có lỗi vi phạm, FastAPI lập tức sinh lỗi RequestValidationError, đóng gói thông tin vị trí lỗi (loc) và lý do (msg) rồi trả về client mã 422 Unprocessable Entity. Nếu tất cả hợp lệ, hàm xử lý Python sẽ được chạy.

Bảng Tra Cứu Nhanh Tham Số (Parameter Cheat Sheet)
Loại tham số Cách khai báo trong FastAPI Ví dụ HTTP request Tính chất đặc trưng
Path Parameter @app.get("/items/{item_id}")
def read(item_id: int = Path(...)) 		GET /items/42 - Luôn nằm trong URL.
- Bắt buộc phải truyền.
- Phù hợp định danh tài nguyên duy nhất.
Query Parameter @app.get("/items")
def read(q: str = Query(None)) 		GET /items?q=laptop - Đứng sau dấu ? trên URL.
- Có thể tùy chọn (optional) hoặc bắt buộc.
- Thường dùng để lọc/tìm kiếm.
Request Body class Item(BaseModel):
    name: str
def create(item: Item) 		POST /items
Body: {"name": "laptop"} 		- Chứa dữ liệu JSON phức tạp.
- Phù hợp cho thao tác ghi (POST/PUT).
- Validate tự động theo Pydantic schema.
Các ràng buộc Validation thông dụng

Bạn có thể import các tham số cấu hình này để ràng buộc điều kiện chặt chẽ hơn cho kiểu dữ liệu số hoặc chuỗi:

gt=0 Số lớn hơn 0 (Greater Than)
ge=1 Số lớn hơn hoặc bằng 1 (Greater than or Equal)
lt=100 Số nhỏ hơn 100 (Less Than)
min_length=2 Chuỗi có độ dài tối thiểu là 2 ký tự
pattern="^[a-zA-Z]+$" Chuỗi phải khớp với biểu thức chính quy Regex
