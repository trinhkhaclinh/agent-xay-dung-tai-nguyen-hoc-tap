Thiết lập HTTP Request
Khai báo trong Python: student_id: int (Kiểu số nguyên bắt buộc)
Ràng buộc: Query(min_length=2, max_length=50)
Ràng buộc: Query(1, ge=1) (Giá trị tối thiểu là 1)
Cấu trúc model: name (chuỗi), email (định dạng email), age (lớn hơn 0)
Kết quả từ Server CHỜ GỬI REQUEST
http://localhost:8000/students/5
