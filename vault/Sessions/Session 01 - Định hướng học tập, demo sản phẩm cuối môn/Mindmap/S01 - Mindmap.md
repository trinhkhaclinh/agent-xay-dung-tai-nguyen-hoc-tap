# Định hướng & Demo sản phẩm cuối môn — Mindmap

- **Định hướng & Demo sản phẩm cuối môn**
  - 1. Lộ trình môn học IT-215
    - 26 buổi kết hợp lý thuyết và thực hành dự án
    - Mục tiêu đầu ra: Thiết kế & Xây dựng RESTful API hoàn chỉnh
  - 2. Chuẩn bị môi trường phát triển
    - Ngôn ngữ: Python 3.10+
    - IDE: VS Code
    - Quản lý mã nguồn: Git
    - Kiểm thử API: Postman
  - 3. Kiến trúc hệ thống 3 tầng
    - Client: Trình duyệt/Mobile App gửi Request
    - Server: FastAPI xử lý logic, trả JSON
    - Database: MySQL lưu trữ dữ liệu bền vững
  - 4. Phương pháp học tập chủ động
    - Thiết kế ngược (Backwards Design): Học đi đôi với làm sản phẩm
    - Kỹ năng sửa lỗi, gỡ lỗi (Debugging) là ưu tiên số một
